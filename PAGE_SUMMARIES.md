# Caption Aotearoa — Page Summaries

> For AI design tools to generate layout variations.

## Home

Hero section with branding and decorative Māori patterns. A "join by code" input lets audience members enter an event code to jump straight in. Below that, a searchable list of event cards (title, date, status, code). Footer links to organiser sign-in.

## Login

Email and password form. Link to sign up.

## Sign Up

Email and password form. Link to log in.

## Create Event

Simple form: event title, description, phrase hints for speech recognition, and an optional date/time. Submitting takes you straight to the presenter view.

## Audience View

**Before the event starts:** A waiting screen with a countdown timer, event title, and description.

**During the event:** A header bar with event title, viewer count, and connection status. A language picker to choose your caption language. A settings panel for text size and high contrast mode. The main area is a scrolling feed of live captions.

## Presenter View

A big microphone button in the centre to start/stop recording. Buttons to pick which language you're speaking in (including a bilingual English + Te Reo option). A live preview of captions as you speak. A QR code button that opens a fullscreen shareable code for the audience to scan and join. Header shows event title and viewer count.

## Dashboard

Header with a "New Event" button and sign out. A list of your events as cards showing title, date, and status (upcoming / live / ended). Click a card to manage that event.

## Event Detail

Shows the event info (title, description, date, code, status). Buttons change depending on status: start captioning if upcoming, open presenter or end event if live, download transcript as PDF if ended. QR code button always available.

## Dashboard Presenter

Same as the presenter view, but with a back button to return to the event detail page.
