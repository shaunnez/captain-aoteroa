# Caption Aotearoa: The Digital Weaver - Export Package This document serves as the comprehensive handoff for coding agents to implement the **Caption Aotearoa** design system. ## 1. Design System Summary: "The Digital Weaver" The visual identity is built on the intersection of Māori tradition (*Raranga*) and modern digital accessibility. ### Core Visual Principles: - **Aotearoa Midnight Canvas:** High-contrast charcoal/black base for maximum readability. - **Aurora Gradient:** Vibrant rainbow accents (from the logo) used for active/live states. - **Cultural Anchors:** Functional use of Koru, Kowhaiwhai, and Poutama motifs. ## 2.
